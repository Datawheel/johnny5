Introduction

.. toctree::

johnny5 was developed as part of the `Pantheon <http://pantheon.media.mit.edu>`_ project by the `Collective Learning <http://macro.media.mit.edu>`_ group at the `MIT Media Lab <https://www.media.mit.edu>`_. 

johnny5 is a collection of tools that combines handwritten rules with machine learning, to easily parse and download data from Wikipedia, Wikidata, and other sources.
